# c--_getting_started

This is a simple example of how to use the C++ API for the [C--]() language.

## Requirements
* [C--]()
* [C++ API]()
* [CMake]()
* [C++ compiler]()
* [C++ standard library]()
* [C++ standard library headers]()
* [C++ standard library runtime]()

## Building
```bash
mkdir build
cd build
cmake ..
make
```

## Running
```bash
./c--_getting_started
```
